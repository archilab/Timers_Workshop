# 02 · Bewegung steuert Servo

**MPU6050** liefert Neigung/Beschleunigung; der **Servo** auf **D9** folgt dem normalisierten Wert. Für den **Timers Workshop** auf dem **Arduino Nano** — **Folkwang_Timers** generiert den Code.

## Hardware

- Nano, MPU6050 (**AD0 → VCC**, Adresse **0x69**), Servo **D9**, I²C **A4/A5**.

## Ablauf

1. `prompt.md` an das GPT senden.  
2. Projekt aus `code/` bauen und flashen.
